Botones

Tanto para enviar formularios como para diseñar menús, los elementos button, a e input son necesarios para una buena interacción con el usuario. Utilizan la clase .button para lograr un estilo y funcionalidad general.

Ancla

Barras de búsqueda

Derivadas de los input, las barras de búsqueda ayudaran al usuario a encontrar lo que necesita. Utilízalas junto a la clase .search-bar para obtener el siguiente resultado.

Sliders

Muestra una selección de elementos con el slider, una forma sencilla y elegante que hará que la página sea más dinámica. Utilizando además la clase .vertical-slider conseguirás que este se oriente de forma vertical.
